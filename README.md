## 🚀 Getting Started (For Beginners)

### Step 1: Install Visual Studio Code

Download and install VS Code:
* VS Code: [Visual Studio Code Download](https://code.visualstudio.com/download?utm_source=chatgpt.com)

### Step 2: Install MinGW Compiler

Download MinGW:
* MinGW: https://sourceforge.net/projects/mingw/
After installation:
1. Open MinGW Installation Manager.
2. Select and install:
   * mingw32-base
   * mingw32-gcc-g++
3. Apply changes and complete installation.
4. Add the MinGW `bin` folder to your Windows PATH(In System environment variable).

Example:
```text
C:\MinGW\bin
```
Verify installation:
```bash
g++ --version
```
If a version number appears, MinGW is installed successfully.

### Step 3: Install VS Code Extensions
Open VS Code → Extensions (`Ctrl + Shift + X`)
Install:
* C/C++
* Code Runner
Code Runner allows you to run C++ files directly from VS Code using the ▶ Run button.

### Step 4: Clone This Repository
```bash
git clone https://github.com/purushottam-m-123/DSA-Daily.git
```

### Step 5: Open the Project in VS Code
```bash
cd DSA-Daily
code .
```

### Step 6: Run Any Program
1. Open any `.cpp` file.
2. Click the **Run Code** button (▶) provided by Code Runner.
3. View the output in the terminal/output window.

Alternatively, use the terminal:

```bash
g++ bubbleSort.cpp -o output
output.exe
```
